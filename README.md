Project Description: This project simulates a simple banking system using Java data structures.
It demonstrates how logical and physical data structures work in real-life scenarios such as banking operations.

Data Structures Used:

(1)LinkedList:Used to store all bank accounts.Allows adding, searching, and updating accounts.

(2)Stack:Used for transaction history.Stores deposit and withdraw operations.

Supports:
  1)push (add transaction)
  2)peek (view last)
  3)pop (undo last action)

(3)Queue:

Used in two cases:
  1)Bill payment queue
  2)Account request queue

(4)Array

Stores 3 predefined accounts (Task 6).
Demonstrates physical data structure.

Program Features:

(1)Bank Menu
  
  1)Submit account request
  2)Deposit money
  3)Withdraw money

(2)ATM Menu

  1)Check balance
  2)Withdraw money

(3)Admin Menu

  1)Process account requests
  2)View all accounts
  3)Manage bill queue
  4)View transaction history
  5)Undo last transaction



Conclusion

In this project, I learned how to use different data structures such as LinkedList, Stack, Queue, and Array.
I understood how they can be applied to simulate real banking operations and how data is stored and processed.
